# Assignment3-ADS

A. PROJECT OVERVIEW:

The project implements and compares 3 algorithms:
1) Selection sorting is a basic simple algorithm that works slowly with large data. The difficulty is O(n^2)
2) Quick sort is one of the most efficient and used sorting algorithms. Difficulty - O(nlogn)
3) Binary search is a highly efficient algorithm for finding elements. Difficulty - O(nlogn)

The purpose of this experiment is to run algorithms and compare their performance in different array sizes and input data.
